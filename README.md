# Wikipedia 
Note:-Django special python web framework you need to your server to view the application after downloading.
## Entry Page
### A page that displays the contents of that encyclopedia entry. User can click on any entry name to be taken directly to that entry page. Side bar search allow the user to search any page and even by its keyword
![image](https://github.com/user-attachments/assets/6f0a3977-48a2-484b-9f66-1f54bd7f2ffb)
## NewPage
### : Clicking “Create New Page” in the sidebar take the user to a page where they can create a new encyclopedia entry.
![image](https://github.com/user-attachments/assets/133e9be7-e240-4ab1-8dc1-6ac20ae4fdab)
## Edit Page
###  On each entry page, the user can click a link to be taken to a page where the user can edit that entry’s Markdown content in a textarea.
